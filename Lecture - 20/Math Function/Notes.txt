We can use "Math" object to solve any Mathematical function/Calculation.

# Some methods of Math :
    -> PI
    -> round
    -> sqrt
    -> abs
    -> min
    -> max
    -> pow
    -> floor
    -> ceil
    -> random
It will access by using dot (".").

**random method :** It will generate any random value from 0 to 1.(1 is not included.)
